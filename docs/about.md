# 关于本书

## 《物演通论》

王东岳先生的哲学巨著，提出"递弱代偿"原理，系统阐述宇宙万物的演化规律。

全书分三卷：
- **卷一 · 自然哲学**：存在论基础，递弱代偿原理
- **卷二 · 精神哲学**：感知、认识论、逻辑演化
- **卷三 · 社会哲学**：社会存在、人类文明与前途
